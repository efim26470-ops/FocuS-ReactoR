# Focus Reactor

Статическое PWA-приложение для GitHub Pages. Сервер и база данных не нужны.

## Публикация на GitHub Pages

1. Создайте новый репозиторий.
2. Загрузите содержимое архива в корень репозитория.
3. Откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите **Deploy from a branch**.
5. Выберите ветку `main` и папку `/root`, затем сохраните.

Через несколько минут приложение будет доступно по адресу GitHub Pages.

## Установка на iPhone

Откройте сайт в Safari → **Поделиться** → **На экран «Домой»** → **Добавить**.

## Возможности

- режимы 25/5, 50/10, 90/20 и пользовательский;
- восстановление активного таймера после закрытия приложения;
- реактор, заряд, стабильность и прерывания;
- блоки целей;
- статистика по дням и часам;
- локально генерируемые фоновые звуки;
- полноэкранный режим;
- блокировка случайных нажатий с удержанием для разблокировки;
- достижения и открываемые реакторы;
- 8 визуальных тем;
- экспорт отчёта и резервной копии;
- офлайн-работа через Service Worker.
