# Focus Reactor

## Установка на iPhone

Откройте сайт в Safari → **Поделиться** → **На экран «Домой»** → **Добавить**.

## Возможности
- режимы 25/5, 50/10, 90/20 и пользовательский;
- восстановление активного таймера после закрытия приложения;
- реактор, заряд, стабильность и прерывания;
- блоки целей;
- статистика по дням и часам;
- локально генерируемые фоновые звуки;
- полноэкранный режим;
- блокировка случайных нажатий с удержанием для разблокировки;
- достижения и открываемые реакторы;
- 8 визуальных тем;
- экспорт отчёта и резервной копии;
- офлайн-работа через Service Worker.
