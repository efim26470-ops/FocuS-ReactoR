# Focus Reactor v2

Focus Reactor — полностью автономное PWA-приложение для концентрации. Проект не использует сервер, базу данных, внешние API, библиотеки или аудиофайлы. Все данные и автоматические резервные копии хранятся локально в браузере.

## Что реализовано

- Живой реактор: температура, давление, энергия, стабильность, перегрев, критическое состояние, сверхзаряд и квантовый резонанс.
- Аварийное охлаждение с ограниченными зарядами и улучшениями.
- Режимы 25/5, 50/10, Deep 90/20, Flow Mode, Boss Session 90–180 минут, Sprint, Night Shift и собственный режим.
- Досрочное завершение сессии: в статистику всегда попадает фактическое время.
- Отчёт после сессии: выполненная работа, лёгкость концентрации, энергия, настроение и заметка.
- Умные перерывы с дыханием, упражнениями для глаз, водой, прогулкой и бонусом стабильности.
- Карта лаборатории из восьми постепенно открываемых комнат.
- Шесть веток исследований с реальными постоянными бонусами.
- Ежедневные и недельные контракты с энергией, опытом, темами, звуками, реакторами, рамками и эффектами ядра.
- Многоуровневые и секретные достижения.
- Проекты с цветом, иконкой, целью, дедлайном, заметками, сериями и отдельной историей.
- Аналитика по часам, дням недели, стабильности, проектам, режимам, темам и звукам.
- 24 визуальные темы, включая OLED, Nuclear Green, Soviet Control Room, Retro Terminal, Cyberpunk, Arctic, Mars, Deep Ocean, Solar, Quantum, Paper, Minimal, Game Boy, Macintosh 1984, Windows 95, Cartoon, Anime, Industrial, Red Alert и Cozy Night Lab.
- Синтетические фоновые звуки через Web Audio API.
- Полноэкранный режим, блокировка случайных нажатий, вибрация и Wake Lock.
- Восстановление активного таймера после закрытия PWA по абсолютному времени.
- Экспорт JSON, CSV, HTML, печать в PDF, статистика без настроек и CSV выбранного проекта.
- Импорт полной копии и локальные автоматические резервные копии.
- Миграция данных из первой версии Focus Reactor без сброса прогресса.

## Публикация на GitHub Pages

1. Создайте новый публичный репозиторий на GitHub.
2. Загрузите **содержимое архива** в корень репозитория. Файл `index.html` должен лежать в корне.
3. Откройте `Settings → Pages`.
4. В разделе `Build and deployment` выберите `Deploy from a branch`.
5. Выберите ветку `main`, папку `/root` и нажмите `Save`.
6. После публикации откройте выданную GitHub Pages ссылку.

## Установка на iPhone

Откройте сайт в Safari, нажмите «Поделиться», выберите «На экран Домой» и нажмите «Добавить». После первого открытия приложение и основные файлы будут доступны офлайн.

## Обновление поверх первой версии

Замените файлы проекта в репозитории содержимым новой версии. При первом запуске приложение попробует прочитать данные из ключа первой версии и автоматически перенесёт историю, цели, серии и настройки.

## Состав проекта

- `index.html` — структура приложения.
- `styles.css` — адаптивный интерфейс и все темы.
- `app.js` — таймер, прогресс, проекты, аналитика, экспорт и игровая логика.
- `manifest.webmanifest` — параметры установки PWA.
- `sw.js` — офлайн-кэш и обновление файлов.
- `icons/` — иконки для iOS и PWA.
- `.nojekyll` — публикация статических файлов GitHub Pages без обработки Jekyll.
